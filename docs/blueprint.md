# **App Name**: LetsCatchUp Learn

## Core Features:

- Interactive Homepage & Landing Pages: Showcase core LMS features, benefits, and call-to-actions (Enroll Now, Learn More) using dynamic, SaaS-style landing page sections inspired by the template. Integrates content from letscatchup-kcs.com.
- Secure User Authentication: Enable secure user registration and login functionality for students and instructors, powered by Firebase Authentication.
- Dynamic Course Catalog: Browse and search a comprehensive catalog of available courses, each with detailed descriptions, curriculum outlines, and instructor information, stored in Firestore.
- Course Enrollment & Management: Allow authenticated users to enroll in courses, view their active courses, and manage their learning path through data stored in Firestore.
- Personalized Learning Progress Tracking: Display a user's progress within each enrolled course, including completed lessons, modules, and overall completion status, with data managed in Firestore.
- AI Course Recommendation Tool: An AI tool that suggests relevant courses and personalized learning paths to users based on their enrolled courses, performance, interests, and profile data, powered by generative AI.
- Contact & Inquiry Submission: Provide a contact form for prospective students and users to submit inquiries or feedback, with submissions securely stored in Firestore for management.

## Style Guidelines:

- Primary action color: A vibrant purple-blue (#4C33CC) reflecting a modern, educational yet approachable tone.
- Background color: A deep, desaturated purplish-blue (#1E1E2E) providing a sophisticated dark theme to emphasize glassmorphism and glowing effects.
- Accent color: A bright, clear cyan-blue (#33CCFF) to provide visual contrast and highlight secondary interactions or complementary information.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-inspired feel. Body font: 'Inter' (sans-serif) for clean readability and objective presentation.
- Utilize soft glowing icons within rounded, often translucent shapes, complementing the dark background and glassmorphism elements.
- Implement a 'dark blue gradient SaaS' inspired layout featuring generous spacing, rounded cards with a glassmorphism effect, distinct hero sections, and clearly defined content blocks for a premium feel.
- Incorporate smooth and subtle hover animations on interactive elements, cards, and buttons to enhance user engagement and visual polish.
